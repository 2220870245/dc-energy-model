# 子任务 5: 实现深度学习主模型与阶段性对比实验

## 目标

在固定数据集和 benchmark 的前提下，实现真正值得比较的深度学习模型，而不是只堆一个复杂网络结构。

## 模型范围
- LSTM 或 GRU
- Transformer、TCN 或 PatchTST 风格模型
- 支持单步预测，后续可扩展到多步预测

## 执行步骤

### Phase 1: LSTM 管道
- 建立时序窗口 dataset
- 实现训练循环、早停、checkpoint 和日志
- 先在小规模数据上确认 loss 下降和推理正确

### Phase 2: Transformer 管道
- 在相同数据切分上实现第二类主模型
- 对齐输入窗口、损失函数和指标
- 控制参数量和训练成本

### Phase 3: 对比和消融
- 对比深度学习模型与最强 baseline
- 做窗口长度、特征子集、历史标签使用方式等消融
- 形成是否继续扩大训练的判断依据

## 阶段性验证

### Gate 1: 训练稳定性验证
通过条件：
- 模型可以完整训练至少一个 epoch
- loss 与指标记录正常
- checkpoint 可加载并复现验证结果

### Gate 2: 公平对比验证
通过条件：
- 使用和 baseline 相同的数据切分与指标
- 至少一个深度学习模型超过最强 baseline 或接近可接受阈值

### Gate 3: 消融验证
通过条件：
- 已完成至少一轮窗口长度或特征消融
- 已能解释性能提升来自哪里，而不是偶然波动

## 交付物
- dataset 与 dataloader 实现
- LSTM 训练配置与日志
- Transformer 或 TCN 训练配置与日志
- 对比实验结果表

## 完成标准
- 至少有一个深度学习模型达到可继续优化的水平
